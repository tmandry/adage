Adage – Classes



Interface


  • TextClass: Simple text, blitted onto the screen at the desired location. Font may be specified.

  • ButtonClass: A 3-Dimensional button, with event support and captions.

  • InputClass: A simple input box, with event support.

  • ScrollBarClass: A scrollbar with event support.

  • ScrollTextClass: A textbox with a scrollbar attached.

  • ScrollInputClass: An input box with a scrollbar attached.

  • ListBoxClass: A listbox with event support.
